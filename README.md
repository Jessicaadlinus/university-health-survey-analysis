# university-health-survey-analysis
This repository contains A statistical analysis project examining the relationship between Body Mass Index (BMI) and blood pressure among university students using real-world survey data.

# Statistical Analysis of BMI and Blood Pressure

## Overview
This project analyzes the relationship between Body Mass Index (BMI) and blood pressure among university students.

## Objective
To determine whether BMI significantly influences blood pressure levels.

## Tools Used
- Microsoft Excel
- Tableau (in progress)

## Project Status
This project is currently in progress. Data cleaning and exploratory analysis have been completed, and dashboard development is ongoing.
